# dotfiles

## Requirements

- [Yolk][]

## Installation

```nushell
git clone https://github.com/phoenixr-codes/dotfiles ~/.config/yolk;
touch ~/.config/yolk/eggs/nushell/secrets.nu;
yolk safeguard
yolk sync
```

You may have to delete some folders such as `~/.config/nushell/`.

## Sync

```console
yolk sync
```

[Yolk]: https://elkowar.github.io/yolk/
